# StudentHub
StudentHub is a web app for Global Academy of Science and Technology where students browse programmes and modules and register their interest. It features a secure admin panel to manage programmes and view student registrations. Built with HTML, CSS, PHP, and a database, it demonstrates full-stack development and CRUD functionality.

📖 Description

StudentHub is a dynamic web application developed for the Global Academy of Science and Technology. It allows students to explore academic programmes and modules, and register their interest in specific programmes. The system is connected to a database for efficient data storage and management.

It also includes a secure admin panel where administrators can manage programmes and view student registrations.

🚀 Features
Browse programmes and modules
Register interest in programmes
Admin login system
Admin dashboard
Add, edit, and delete programmes (CRUD)
View registered students
Database integration
🛠️ Technologies Used
HTML5
CSS (Grid & Flexbox)
PHP
MySQL (Database given by the instituition)
⚙️ How to Run the Project
Clone or download the repository
Place the project folder in your local server (e.g. XAMPP htdocs)
Import the provided database into phpMyAdmin
Start Apache and MySQL

Open your browser and go to:
http://localhost/your-folder-name
